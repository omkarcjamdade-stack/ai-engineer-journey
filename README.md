# AI Sentiment Analyzer

A sentiment analysis pipeline built with Python and HuggingFace Transformers.

## What it does
- Analyzes customer reviews as POSITIVE or NEGATIVE
- Shows confidence score for each prediction
- Saves results to CSV
- Visualizes results as pie and bar charts

## Technologies Used
- Python
- HuggingFace Transformers
- Matplotlib
- CSV

## How to run
pip install transformers torch matplotlib
python day7.py
